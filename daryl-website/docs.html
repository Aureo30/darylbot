Commandes de bases

d$help : Afficher la liste des commandes disponibles

d$ping : Jouer au ping pong !

d$ServerInfos : Affiche les informations du serveur

d$welcome : Souhaite la bienvenue aux nouveaux membres

Modération

d$ban MEMBRE RAISON : Bannir un membre

d$kick MEMBRE RAISON : Expulser un membre

d$clear NUMBER : Effacer des messages

d$rename MEMBRE PSEUDO : Renommer un membre

d$role_add MEMBRE RÔLE : Donner un rôle à un membre

d$role_remove MEMBRE RÔLE : Enlever un rôle à un membre

Fun

d$cat : Affiche une image aléatoire de chat

d$dog : Affiche une image aléatoire de chien

d$gif MOT : Affiche un gif en rapport avec le mot entré
